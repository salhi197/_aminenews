== Description ==

This plugin helps to simplify WordPress admin-menu by hiding the rarely used admin-menu items/links.
If you need to see those menu items, just toggle those menu items with 'Toggle Menu Items' button from the left menu.

== Installation ==

1. Download and activate the plugin.
2. Browse to `Settings > WP Clean Admin Menu` to configure.

== Frequently Asked Questions ==

= How does it work =

It simply hides&show the elements via CSS class.
